# Pharma Sales Data Analysis & Visualization

## 📌 Project Overview
This project analyzes two years (2015 & 2016) of pharma sales data to uncover insights across regions, customer tiers, and time periods.

## 🎯 Objectives
- Compare sales performance (2015 vs 2016)
- Analyze regional and tier-wise contributions
- Identify declining regions and divisions
- Perform quarterly sales analysis
- Visualize insights using bar charts and pie charts

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

## 📊 Key Analysis Performed
- Region-wise sales comparison
- 2016 sales contribution by region (Pie Chart)
- Region & Tier sales breakdown
- Decline detection (states & divisions)
- Quarter-wise sales comparison
- Quarter-wise tier composition (Pie Charts)

## 📈 Key Insights
- Certain regions showed strong growth from 2015 to 2016
- High-tier customers contributed the most to revenue
- Some states and divisions showed a decline, indicating performance gaps
- Seasonal trends observed across quarters

This project demonstrates end-to-end exploratory data analysis, data transformation, and visualization to extract meaningful business insights.
